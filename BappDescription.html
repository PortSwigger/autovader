AutoVader is a Burp Suite extension that integrates DOM Invader with Playwright Java to automatically find DOM-based vulnerabilities in web applications. It provides automated scanning capabilities through context menu actions, allowing security testers to quickly identify client-side security issues without manual DOM Invader interaction.

Features

  • Automated DOM vulnerability scanning: No manual DOM Invader interaction needed.
  • Multiple scan profiles: Optimized settings for different vulnerability types.
  • Issue deduplication: Prevents duplicate findings across scans.
  • Project-specific canaries: Unique identifiers per Burp project.
  • Burp integration: Reports findings directly as Burp Scanner issues.
  • Configurable payloads: Customize injection payloads via settings.

Usage

AutoVader adds a context menu to Burp Suite with multiple scanning options. You can right-click on any request in the Target, Proxy History, or Repeater tabs to access these features:

  • Open DOM Invader: Opens a browser with DOM Invader configured for manual testing. Useful for debugging or custom analysis.
  • Scan all GET params: Automatically enumerates all query parameters, injects canary values into each parameter, and detects DOM-based vulnerabilities from URL inputs.
  • Scan for client-side prototype pollution: Detects prototype pollution vulnerabilities, tests query string, hash, and JSON inputs, and verifies pollution with automated checks.
  • Intercept client-side redirect: Sets breakpoints on client-side redirects to help identify open redirect vulnerabilities.

AutoVader also provides project-specific settings accessible through Burp Suites Settings, where you can overwrite DOM Invader and Chromium paths, customize payload to append to canary values when scanning, scan specific tags for gadgets, set delay between requests, control viewing of devtools panel, enable or disable headless scanning, and autotrigger AutoVader for requests from Repeater, Intruder or another extension.

AutoVader then uses Playwright to launch a headless Chromium browser with DOM Invader extension, configures DOM Invader settings based on the selected scan type, navigates the browser to target URLs with appropriate payloads, captures DOM Invader findings via callbacks, reports them as Burp issues, and finally deduplicates issues to prevent duplicate findings.

AutoVader leverages Playwright Java for browser automation, DOM Invader for vulnerability detection, and Burp Montoya API for extension integration. The extension automatically detects Burps bundled Chromium installation and the DOM Invader extension paths, requiring no manual configuration.
